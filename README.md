
# My .NET App starter template

Very simple ToDo app to use as a starter template for build applications

## Requirements

- .NET CLI v8
- .NET SDK v8
- Node.js v18+

You can also use [Github Codespaces](https://codespaces.new/jsteinshouer/dotnet-starter).

## Getting Started

### Backend

```
cd ToDoApi
dotnet run
```

### Front-end

```
cd ToDoApp
npm install
npm run start
```
